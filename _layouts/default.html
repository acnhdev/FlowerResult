Not all flowers of same species and color are equal!! They might look alike but their genes can be different, and this will affect the offspring color and rate when being crossbred.

For example, there is a total of 13 red rose variant. If you crossbreed 2 red roses, depending on the gene, the offspring they produce may be different.

The table below shows list of flowers and number of variants available for each color.

To keep track of flower gene, it is best that you start from seed as the gene of seed is fixed.

DEFINING PARENT

Key in the two character hex value of the flowers (e.g. 00, 4f, ..)

I have also created label for each variant for me to keep track of my flowers. You may use it to define the parents too. It is a 3 characters string, the first being the color code (refer table) followed by two digits number.

For example, r04 means red variant 4, u01 means purple variant 1 etc.

Seed color will always be represented by number one e.g. r01 is red seed, y01 is yellow seed.

If you key in 'r01' for parent A and 'r01' for parent B, you will get the result for crossbreeding 2 red seed flower.
