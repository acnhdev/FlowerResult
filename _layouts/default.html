Key in label or hex value for each parent.

I have assigned each color variant with a label code for me to keep track of my flower. It is a 3 characters string, the first being the color code (refer table) followed by two digits running number.

For example, r04, u01, y14, h02, ...

Seed color will always be represented by number one e.g. r01 is red seed, y01 is yellow seed.
